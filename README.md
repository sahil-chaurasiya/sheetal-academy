## 🚀 Local Development

Clone the repository:

```bash
git clone https://github.com/sahil-chaurasiya/sheetal-academy.git
cd sheetal-academy
```

Install dependencies:

```bash
npm install
# or
yarn
# or
pnpm install
```

Start the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open in browser:

```text
http://localhost:3000
```
